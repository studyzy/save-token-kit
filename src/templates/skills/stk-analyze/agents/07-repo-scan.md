# 子 Agent 7: 仓库扫描 (repo-scan)

**输入**：`repo-scan.json` + `context.json`
基于扫描结果给出仓库级建议：

| 条件                                 | 输出                                                                      |
| ------------------------------------ | ------------------------------------------------------------------------- |
| `sameRepo=same` 且 `docFileCount` 大 | 建议排除文档目录出自动上下文。`action`: "在 CODEBUDDY.md 排除 docs/ 目录" |
| `isMonorepo`                         | 建议按子包加载上下文                                                      |

**输出 category**：`仓库专项`
**`level`**：中级（仓库配置优化类）
